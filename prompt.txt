## Cursor Master Prompt

You are a senior Laravel enterprise architect and UI/UX engineer.

Create an **enterprise-grade, modular government-ready information system** using **Laravel 12**, **PostgreSQL**, **Blade**, and **Bootstrap 5**.

This system must be designed as a **base platform** that can support many future modules without refactoring. The **Case Management System (CMS)** must be implemented as **Module 1**, built on top of this base platform.

---

Also Read Related Files in this projct folder

## 1. Core Platform Architecture

Build a clean, scalable, modular architecture suitable for long-term enterprise use.

### Key Principles

* Laravel 12 best practices
* PostgreSQL-first design
* Modular architecture (plug-and-play modules)
* Service-layer business logic
* Policy-based authorization
* Auditability and compliance readiness

### Directory Structure

```
app/
  Core/
    Auth/
    RBAC/
    Audit/
    Settings/
    Dashboard/
Modules/
  CaseManagement/
    Controllers/
    Models/
    Services/
    Requests/
    Policies/
    Migrations/
    Views/
    Routes/
```

Each module must:

* Register its own routes
* Define its own permissions
* Expose menu items dynamically
* Be removable without breaking the core system
